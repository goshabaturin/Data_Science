Тут содержатся различные проекты выполненные в рамках курса Data Science от Яндекс.Практикум. Проекты разбиты на 5 блоков:

- Выпускной проект

- Машинное обучение

Для удобства, описания проектов из каждого блока были собраны в таблицы и снабжены гиперссылками на notebook'и с решениями поставленных задач.
Стоит отметить, что несмотря на то, что был выделен отдельный блок "Машинное обучение", фактичести, читатель может встретить решение задач машинного обучения в каждом из перечисленных блоков.
